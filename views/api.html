AirlineStats API:

Endpoints:

  • api/flights

    • Accepted Methods:

    • GET

    Headers:

    From

    • REQUIRED
    • Type: [ AirlineCode ]
    • Description: Location of departure

    To

    • REQUIRED
    • Type: [ AirlineCode ]
    • Description: Location of arrival

    Departure Date

    • REQUIRED
    • Type: [ Date ( YYYY-MM-DD ) ]
    • Description: Date of departure

    Arrival Date

    • REQUIRED
    • Type: [ Date ( YYYY-MM-DD ) ]
    • Description: Date of arrival

    Round Trip

    • Default = false
    • Type: [ Boolean ]
    • Description: Filter only round trip flights

    Occupants

    • Default = 1
    • Type: [ Integer ]
    • Description: Amount of occupants

    Stops

    • Default = false
    • Type: [ Boolean ]
    • Description: Allow flights with stops

    Class

    • Default = 1
      • 1 = Economy
      • 2 = Premium economy
      • 3 = Buisness / First
    • Type: [ Integer ]
    • Description: Class of flight

    Sort

    • Default = 1
      • 1 = Sort by price
      • 2 = Sort by flight duration
    • Type: [ Integer ]
    • Description: Sort results
